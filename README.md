# Руководство по написанию сценариев Carrot

Данное руководство было подготовлено для тех, кто хочет автоматизировать процесс работы с шаблонами и управлять их свойствами.

**Carrot** работает с проектами, которые были созданы в **After Effects** и экспортированы как готовые **шаблоны**. Написанные сценарии, основанные на языке JavaScript, в свою очередь, позволяют управлять содержимым шаблона программно - без необходимости настройки шаблонов вручную.

## Зачем нужны сценарии?

Сценарии помогают автоматизировать рутинные процессы, то есть вместо того, чтобы каждый раз выполнять одни и теже действия, Вы можете написать код один раз - и он сделает всю работу за Вас. Это особенно полезно, когда используется какая-либо **"многосвязанная"** графика, где определённое событие должно запускаться по какому-либо условию.

## Как построено это руководство?

Мы постарались собрать здесь всё, что нужно, чтобы начать и уверенно продолжить писать сценарии. Сначала расскажем как всё устроено, потом покажем как запускать скрипты, где их хранить и как производить отладку. Дальше - полный справочник по всем доступным объектам, методам и свойствам шаблонов и как итог - различные примеры кода, которые часто могут встречаться в работе.

## Кому это пригодится?

Руководство написано для всех, кто уже мог сталкиваться с шаблонами **Carrot** или только начинает: моушн-дизайнеров, технических специалистов, разработчиков автоматизаций и других специлистов. Если Вы хоть немного знаете JavaScript - будет проще. Если нет - не страшно, примеры понятные и можно будет использовать их сразу.

Мы старались объяснить всё максимально просто, без воды и без сложных терминов там, где можно обойтись обычнымми словами.
